#File List

1. gmbrd.cpp, gmbrd.hpp - Define the game board class
2. tic_tac_toe.cpp - Define the game logics and run the actual game
3. training.cpp - Train the AI
4. weight.txt - Hold the weights
5. trace.txt - Record the training result
6. startingweights.txt - Hold the start weights before the training
7. revisedweights.txt - Hold the revised weights after training
8. control.txt - Hold the training configurations

#文件说明

1. gmbrd.cpp, gmbrd.hpp - 定义棋盘类
2. tic_tac_toe.cpp - 定义游戏逻辑，执行游戏
3. training.cpp - 训练AI
4. weight.txt - 保存权重值
5. trace.txt - 记录每一次的训练误差结果
6. startingweights.txt - 训练前的起始权重值
7. revisedweights.txt - 训练后的矫正权重值
8. control.txt - 训练程序的设置文件
